# 镜览 / aReader

<h3 align="center">🚸Electron注意</h3>

<p align="center">这个应用使用了<b>Electron</b></p>

# Feature

- 📚书架
  - 最近阅读
    - 添加到书架
  - 导入书籍
  - 移除书籍
  - 分类
  - 搜索
  - 排序
  - 显示模式
  - 进度
- 📖阅读
  - 目录
    - 书签
  - 阅读模式
    - 滚动模式
    - 分页模式
  - 字体
    - 字体类型
    - 字体大小
    - 首行缩进
    - 行高
  - 颜色配置
    - 基础方案
    - 文本颜色
    - 背景颜色
  - 全文检索
  - 进度
  - 跳转
- 💡格式支持
  - EPUB
  - TEXT
